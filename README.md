## Installation

1. Clone the repository:
    git clone https://github.com/amithkumarhm/my-portfolio.git

2. Navigate to the project directory:
    cd my-portfolio

3. Install dependencies:
    npm install

4. Start the development server:
    npm run dev

5. Open your browser and visit:
    http://localhost:5173
